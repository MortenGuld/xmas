# Santa Hat Toss

A fun Christmas mini-game where you toss Santa hats onto the leadership team!

**Play now:** https://mortenguld.github.io/xmas/

## How to Play

1. Click anywhere to start the music
2. Grab a falling Santa hat by clicking/dragging it
3. Drag and release to throw the hat at a team member's ornament
4. Place hats on all 10 team members to win!

### Controls

- **Mouse**: Drag and throw hats
- **R**: Restart game
- **M**: Toggle music/sound
- **ESC**: Drop held hat

## Run Locally

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## Tech Stack

- TypeScript + Vite
- GSAP for animations
- Canvas Confetti for celebrations
- Web Audio API for sound effects
